JavaScript Arrays

Array

An array in JavaScript is a special object that stores multiple values in a single variable. You can store numbers, strings, booleans, or even other arrays or objects!

         
          let fruits = ["Apple", "Banana", "Mango"];

          fruits.push("Orange");       // Add an element
          console.log(fruits[0]);      // Access first element
          fruits[1] = "Grapes";        // Change second element
          
          console.log(fruits);         // Output the whole array
          Apple
          ["Apple", "Grapes", "Mango", "Orange"]

Array methods

1. Array.length

Gets the number of elements in an array.


          let fruits = ["Apple", "Banana", "Cherry"];
          console.log(fruits.length);  // Output: 3

2. Array.tostring()

Converts the array into a comma-separated string. 


          let fruits = ["Apple", "Banana", "Cherry"];
          console.log(fruits.toString());  // Output: Apple,Banana,Cherry

3. Array.at()

Returns the element at a specific index (supports negative indexing). 


          let fruits = ["Apple", "Banana", "Cherry"];
          console.log(fruits.at(1));   // Output: Banana
          console.log(fruits.at(-1));  // Output: Cherry

4.Array.join()

Joins all elements into a string with a custom separator.


          let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits.join(" - "));  // Output: Apple - Banana - Cherry

5.Array.pop()

Removes and returns the last element.


          let fruits = ["Apple", "Banana", "Cherry"];
          fruits.pop();
          console.log(fruits);  // Output: ["Apple", "Banana"]

6.Array.push()

Adds new elements to the end.


        let fruits = ["Apple", "Banana"];
        fruits.push("Mango");
        console.log(fruits);  // Output: ["Apple", "Banana", "Mango"]

7.Array.shift()

Removes and returns the first element.


      let fruits = ["Apple", "Banana", "Cherry"];
      fruits.shift();
      console.log(fruits);  // Output: ["Banana", "Cherry"]

8.Array.unshift

Adds elements to the beginning.


    let fruits = ["Banana", "Cherry"];
fruits.unshift("Apple");
console.log(fruits);  // Output: ["Apple", "Banana", "Cherry"]

9.Array.delete()

Deletes an element but leaves an empty slot (undefined).


    let fruits = ["Apple", "Banana", "Cherry"];
delete fruits[1];
console.log(fruits);  // Output: ["Apple", undefined, "Cherry"]

10.Array.concat()

Merges two or more arrays.


    let fruits = ["Apple", "Banana"];
let moreFruits = ["Cherry", "Mango"];
let allFruits = fruits.concat(moreFruits);
console.log(allFruits);  // Output: ["Apple", "Banana", "Cherry", "Mango"]

11.Array.copyWithin()

Copies part of an array to another location in the same array.


    let arr = [1, 2, 3, 4, 5];
arr.copyWithin(0, 3);  
console.log(arr);  // Output: [4, 5, 3, 4, 5]

12.Array.flat()

Flattens nested arrays.


    let arr = [1, [2, [3, 4]]];
    console.log(arr.flat(2));  // Output: [1, 2, 3, 4]

13.Array.slice()

Extracts a portion of an array (without modifying it).


    let fruits = ["Apple", "Banana", "Cherry", "Mango"];
    console.log(fruits.slice(1, 3));  // Output: ["Banana", "Cherry"]

14.Array.splice()

Adds/removes elements at a specified index (modifies the array).


    let fruits = ["Apple", "Banana", "Cherry"];
fruits.splice(1, 1, "Mango");  // Remove 1 item at index 1, add "Mango"
console.log(fruits);  // Output: ["Apple", "Mango", "Cherry"]

15.Array.toSpliced()

Same as splice(), but does not modify the original array.


    let fruits = ["Apple", "Banana", "Cherry"];
let newFruits = fruits.toSpliced(1, 1, "Mango");
console.log(newFruits);  // Output: ["Apple", "Mango", "Cherry"]
console.log(fruits);     // Original remains unchanged

Array Search Methods

1.Array.indexOf()

Finds the first index of a given element in an array.


    let fruits = ["apple", "banana", "mango", "banana"];
    let position = fruits.indexOf("banana");
    
    console.log(position); // Output: 1

     >Returns 1 because "banana" first appears at index 1.
     >Returns -1 if the element isn't found.

2.Array.lastIndexOf()

Finds the last index of a given element.


    let fruits = ["apple", "banana", "mango", "banana"];
    let position = fruits.lastIndexOf("banana");
    
    console.log(position); // Output: 3

    >Returns 3 (the last "banana" is at index 3).

3.Array.includes()

Checks if an element exists in the array.


    let fruits = ["apple", "banana", "mango"];

console.log(fruits.includes("banana")); // true
console.log(fruits.includes("grape"));  // false


    >Returns a Boolean (true or false).

4.Array.find()

Returns the first element that matches a condition.


    let numbers = [4, 9, 16, 25, 29];
let result = numbers.find(num => num > 20);

console.log(result); // Output: 25



    >Stops searching after finding the first match.

5.Array.findIndex()

Returns the index of the first element that matches a condition.


    let numbers = [4, 9, 16, 25, 29];
    let index = numbers.findIndex(num => num > 20);
    
    console.log(index); // Output: 3

6.Array.findLast()

Returns the last element that matches a condition.


    let numbers = [4, 9, 16, 25, 29];
let result = numbers.findLast(num => num > 20);

console.log(result); // Output: 29

7.Array.findLastIndex()

Returns the index of the last element that matches a condition.


    let numbers = [4, 9, 16, 25, 29];
let index = numbers.findLastIndex(num => num > 20);

console.log(index); // Output: 4

javascript array sort

Alphabetic Sort

1.Array.sort()

This method sorts the elements of an array in place (meaning it changes the original array).


    const fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.sort();
console.log(fruits);
// Output: ["Apple", "Banana", "Mango", "Orange"]

2.Array.reverse()

This method simply reverses the order of the elements in an array. It doesn't sort, it just flips the array. It also modifies the original array.


    const letters = ['a', 'b', 'c', 'd'];
    letters.reverse();
    console.log(letters);
    // Output: ['d', 'c', 'b', 'a']

3.Array.tosorted()

This is a new method. It works just like sort(), but it does not change the original array. Instead, it returns a new, sorted array. This is safer to use.


    const months = ["Mar", "Jan", "Feb"];
const sortedMonths = months.toSorted();

console.log(sortedMonths);
// Output: ["Feb", "Jan", "Mar"]

console.log(months);
// Output: ["Mar", "Jan", "Feb"]  <-- The original is unchanged!

4.Array.toReversed

This is the new version of reverse(). It does not change the original array and returns a new, reversed array.


    const nums = [1, 2, 3, 4];
const reversedNums = nums.toReversed();

console.log(reversedNums);
// Output: [4, 3, 2, 1]

console.log(nums);
// Output: [1, 2, 3, 4]  <-- The original is unchanged!

5.Sorting Objects

This is the new version of reverse(). It does not change the original array and returns a new, reversed array.


    const cars = [
  {type: "Volvo", year: 2016},
  {type: "Saab", year: 2001},
  {type: "BMW", year: 2010}
];

// Sort by year (numerically)
cars.sort(function(a, b) {
  return a.year - b.year;
});

console.log(cars);
/* Output:
[
  {type: "Saab", year: 2001},
  {type: "BMW", year: 2010},
  {type: "Volvo", year: 2016}
]
*/

Numeric Sort

> The compare function takes two values (a and b) and returns a number. For ascending order (smallest to largest), use: (a, b) => a - b For descending order (largest to smallest), use: (a, b) => b - a 


    const points = [40, 100, 1, 5, 25, 10];

// Default (incorrect) string sort:
// [1, 10, 100, 25, 40, 5]
points.sort();

// Correct numeric sort (Ascending)
points.sort((a, b) => a - b);
// Result: [1, 5, 10, 25, 40, 100]

// Correct numeric sort (Descending)
points.sort((a, b) => b - a);
// Result: [100, 40, 25, 10, 5, 1]

1.Random Sort (Shuffle)

This uses the compare function to randomize the array order.


    const points = [40, 100, 1, 5, 25, 10];
points.sort(() => 0.5 - Math.random());
// Result: [25, 10, 1, 100, 5, 40] (or any random order)

2.Math.min() and Math.max()

These functions find the single lowest or highest value in an array. They do not sort the array. You must use the spread (...) operator to pass the array elements as arguments.


    const points = [40, 100, 1, 5, 25, 10];

// Find the lowest value
let minVal = Math.min(...points); // Result: 1

// Find the highest value
let maxVal = Math.max(...points); // Result: 100

3."Home made" Min() and Max()

This refers to writing your own function (usually with a for loop) to find the minimum or maximum value. This is a common exercise to learn programming logic.


    function findMax(arr) {
      let len = arr.length;
      let max = -Infinity; // Start with the lowest possible value
    
      while (len--) {
        if (arr[len] > max) {
          max = arr[len];
        }
      }
      return max;
    }
    
    const points = [40, 100, 1, 5, 25, 10];
    let myMax = findMax(points); // Result: 100

Array Iteration Methods

1.Array.forEach()

The forEach() method executes a provided function once for each array element. Goal: To perform an operation (a side effect) for every item, such as logging, updating a database, or modifying an external variable. Return Value: It returns undefined. It does not create a new array. Example This example logs a message for every item in the array.


      const colors = ['red', 'green', 'blue'];

colors.forEach(item => {
  console.log(`The color is: ${item}`);
});

// Output:
// The color is: red
// The color is: green
// The color is: blue

2.Array.map()

The map() method creates a new array populated with the results of calling a provided function on every element in the calling array. Goal: To transform every element in the array into a new value. Return Value: A new array with the same length as the original. The original array remains unchanged.


    const numbers = [1, 2, 3, 4];

const doubledNumbers = numbers.map(num => {
  return num * 2;
});

console.log(doubledNumbers); // [2, 4, 6, 8]
console.log(numbers);       // [1, 2, 3, 4] (Original array is unchanged)

3.Array.filter()

The filter() method creates a new array containing all elements that pass a test (a condition that returns true or false) implemented by the provided function. Goal: To select a subset of elements from an array based on a condition. Return Value: A new array with elements that met the condition. The new array's length will be less than or equal to the original.


    const scores = [5, 12, 8, 25, 3];

const highScores = scores.filter(score => {
  return score > 10;
});

console.log(highScores); // [12, 25]

4.Array.flatMap()

The flatMap() method is a combination of map() and flat(1). It's used to map each element and then flatten the resulting structure by one level in a single step.


  const phrases = ["Quick brown fox", "jumps over", "the lazy dog"];

  // flatMap is equivalent to map() followed by flat(1)
  const allWords = phrases.flatMap(phrase => {
    return phrase.split(' ');
  });
  
  console.log(allWords);
  // Output: ['Quick', 'brown', 'fox', 'jumps', 'over', 'the', 'lazy', 'dog']

Array Const

In JavaScript, const declares a constant reference (or constant pointer) to a value. You CANNOT reassign the array: You cannot change the variable myArray to point to a completely different array or value. You CAN modify the array content: Because const only protects the reference, you are free to change the elements inside the array (e.g., adding, removing, or changing elements).


      // 1. Declare the array using const
const fruits = ["apple", "banana", "cherry"];
console.log(fruits); // Output: ["apple", "banana", "cherry"]

// --- ACTIONS THAT ARE ALLOWED (Modifying the array content) ---

// A. Changing an element at an index is allowed
fruits[0] = "orange";
console.log(fruits); // Output: ["orange", "banana", "cherry"]

// B. Adding new elements (mutation) is allowed
fruits.push("grape");
console.log(fruits); // Output: ["orange", "banana", "cherry", "grape"]

// --- ACTIONS THAT ARE NOT ALLOWED (Reassigning the variable) ---

// C. Reassigning the variable will throw an error:
// fruits = ["kiwi", "lime"]; 
// console.log(fruits); 
// TypeError: Assignment to constant variable.
