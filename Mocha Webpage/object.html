JavaScript Objects

1. What is an Object?

In JavaScript, an object is a collection of properties, written as key-value pairs. The key is a string (property name), and the value can be any data type. 

let person = {
  name: "Rani",
  age: 20,
  city: "Pune"
};

console.log(person);

2. Object Properties

Object properties store data as key-value pairs. You can access, add, modify, or delete them. 

let person = {
  name: "Rani",
  age: 20,
  city: "Pune"
};

console.log(person.name);
console.log(person["city"]);

person.country = "India";
person.age = 21;
delete person.city;

console.log(person);

3. Object Methods

A method is a function stored as a property of an object. 

let student = {
  firstName: "Rani",
  lastName: "Roge",
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};

console.log(student.fullName());

4. Displaying Objects

You can display object data in multiple ways. 

let car = { brand: "Tata", model: "Nexon", year: 2023 };

// Using dot notation
console.log(car.brand);

// Using loop
for (let key in car) {
  console.log(key + ": " + car[key]);
}

// Convert to JSON
console.log(JSON.stringify(car));

5. Adding and Modifying Properties

Objects in JavaScript are dynamic â€” you can change them anytime. 

let mobile = {
  brand: "Samsung",
  model: "S21"
};

mobile.price = 70000;
mobile.model = "S22 Ultra";

console.log(mobile);

6. The this Keyword

Inside an object method, this refers to the object itself. 

let laptop = {
  brand: "HP",
  model: "Pavilion",
  showDetails: function() {
    console.log(this.brand + " " + this.model);
  }
};

laptop.showDetails();

7. Nested Objects

Objects can contain other objects as properties.

let employee = {
  name: "Rani",
  address: {
    city: "Pune",
    pincode: 411001
  }
};

console.log(employee.address.city);

8. Array of Objects

Store multiple objects in an array.

let students = [
  { name: "Rani", age: 20 },
  { name: "Asha", age: 21 },
  { name: "Meena", age: 19 }
];

console.log(students[1].name);

9. Built-in Object Methods

Common JavaScript object methods include:

let car = { brand: "Maruti", model: "Swift", year: 2024 };

console.log(Object.keys(car));
console.log(Object.values(car));
console.log(Object.entries(car));

10. Object.assign()

Used to merge two or more objects.

let obj1 = { a: 1, b: 2 };
let obj2 = { b: 3, c: 4 };

let merged = Object.assign({}, obj1, obj2);
console.log(merged);

11. Object.freeze()

Prevents adding, deleting, or changing properties.

let user = { name: "Rani", age: 20 };
Object.freeze(user);

user.age = 25; // No effect
user.city = "Pune"; // No effect
console.log(user);

12. Object.seal()

Prevents adding or deleting, but allows modifying properties.

let phone = { brand: "Samsung", model: "S21" };
Object.seal(phone);

phone.model = "S22"; // Works
phone.price = 70000; // Fails
console.log(phone);

13. Object Destructuring

Extract specific properties into variables.

let book = {
  title: "JavaScript Basics",
  author: "Rani Roge",
  year: 2025
};

let { title, author } = book;
console.log(title);
console.log(author);

14. Spread Operator with Objects

Copy or merge objects using ... operator.

let obj1 = { a: 1, b: 2 };
let obj2 = { c: 3, d: 4 };

let combined = { ...obj1, ...obj2 };
console.log(combined);

15. Comparing Objects

Objects are compared by reference, not by value.

let objA = { name: "Rani" };
let objB = { name: "Rani" };

console.log(objA == objB);  // false
console.log(objA === objB); // false
