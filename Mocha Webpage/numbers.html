JavaScript Numbers

1. Number introduction

In JavaScript, numbers are used to represent both integers and floating-point (decimal) values. Unlike many other programming languages, JavaScript has only one type of number, which means there’s no separate type for integers or decimals. Special numeric values include: -Infinity → Represents infinity (e.g., result of 1/0) -Infinity → Negative infinity -NaN → Not-a-Number (e.g., result of invalid math operations like "text" / 2)

let x = 10;           // integer
let y = 5.5;          // floating-point number
console.log(x + y);   // Output: 15.5

2.Number methods

JavaScript provides several built-in methods to work with numbers, convert, format, and check numeric values.

1. Number(): Converts a value to a number.

example: Number("10");  
output:  // 10

example: Number(true);  
output: // 1

example: Number("10.5"); 
output: // 10.5

2. parseInt(): Converts a string to an integer (whole number).

example: parseInt("10.99"); 
output: // 10

example: parseInt("100px"); 
output: // 100

3. parseFloat(): Converts a string to a floating-point number (decimal). 

example: parseFloat("10.99"); 
output: // 10.99

example: parseFloat("5.5em");
output:  // 5.5

4. sNaN(): Checks whether a value is Not-a-Number (NaN).

example: isNaN("hello");  
output: // true

example: isNaN(123);      
output: // false

5. isFinite(): Checks if a value is a finite number (not Infinity or NaN).

example: isFinite(10);       
output: // true

example: isFinite(Infinity); 
output: // false

6. toFixed(): Formats a number to a specific number of decimal places.

    example: et num = 5.6789;
            num.toFixed(2); 

output: // "5.68"

7. toPrecision(): Formats a number to a specific length (total digits)..

    example: let n = 123.456;
            n.toPrecision(4);

    output: // "123.5"

8. toExponential(): Returns a number in exponential (scientific) notation.

example: let num = 25;
              num.toExponential(2); 

    output: // "2.50e+1"

9. valueOf(): Returns the primitive value of a Number object.

example: let x = new Number(10);
             x.valueOf(); 

output: // 10

10. Number.isInteger(): Checks if the value is an integer.

example: Number.isInteger(10);   
output: // true

example: Number.isInteger(10.5); 
output: // false

11.Number.isNaN(): Checks if a value is NaN (more reliable than global isNaN()).

example: Number.isNaN(NaN); 
output: // true

example: Number.isNaN("a"); 
output: // false

12. Number.isFinite(): Checks if a value is finite (number only).

example: Number.isFinite(10);      
output: // true

example: Number.isFinite("10");     
output: // false

exam[le: Number.isFinite(Infinity); 
output: // false

13.Number.parseInt(): Same as global parseInt(), but used as a method of Number.

example: Number.parseInt("50px"); 
output: // 50

14. Number.parseFloat(): Same as global parseFloat(), but used as a method of Number.

    example: Number.parseFloat("12.34m"); 
    output: // 12.34

15. Number.toString(): Converts a number to a string.

    example: let x = 10;
            x.toString();
    output: //"10"
