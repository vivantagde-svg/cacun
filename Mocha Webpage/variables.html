JavaScript Variables

• A variable is used to store data or values.
• It works like a container to hold information.
• We can store numbers, text, or other data types.
• Variables help us reuse and update data easily.
• variables can be declared in 4 ways:
a.Let 

b.Const 

c.Var 

d.Automatically

* Identifiers

Identifiers are names used to identify variables, functions, and objects.

The rules for constructing names (identifiers) are:

• Names can contain letters, digits, underscores, and dollar signs.
• Names must begin with a letter, a $ sign or an underscore (_).
• Names are case sensitive (X is different from x).
• Reserved words (JavaScript keywords) cannot be used as names.

* Underscore (_)

The underscore (_) is a valid character that can be used in: 

a) Variable names

b) Function names

c) Object properties


Example:
let total_amount = 5000;
let paid_amount = 2000;
let remaining_amount = total_amount - paid_amount;
console.log("Remaining Amount: " + remaining_amount);

Output:
Remaining Amount: 3000

* Dollar Sign $

The Dollar sign ($) in JavaScript is a valid identifier that can be used in variable names, function names, and object names. It is also used inside template literals to insert (interpolate) variables or expressions into strings.

Example:
let item = "Laptop";
let price = 50000;
console.log(`The price of ${item} is ₹${price}.`);

Output:
The price of Laptop is ₹50000.

Types Of Variable

* Let

• let is used to declare a variable in JavaScript.

• A variable declared with let can be changed later (reassigned).

• let has block scope — it works only inside the { } block where it is defined.

• You cannot redeclare the same let variable in the same block. 

Example:
let name="Paurnima Bawane";
console.log(name);


Output
Paurnima Bawane

* Var

var is used to declare a variable.

• It has function scope (not block scope).

• You can redeclare and reassign a variable declared with var.

• Variables declared with var are hoisted (they move to the top of their scope).

• var is older — now mostly replaced by let and const in modern JavaScript.

Example:
var city = "Nagpur"; 
console.log(city);a


Output:
Nagpur

* Const

• const is used to declare a constant variable.

• You cannot change (reassign) its value after declaring.

• You cannot redeclare a const variable.

• It has block scope (works only inside { }).

• You must assign a value at the time of declaration.

Example:
const country = "India"; 
console.log(country);


Output:
India

Variable Of Datatype

* String

• A string in JavaScript is a sequence of characters used to represent text.

• It is written inside single quotes (' '), double quotes (" "), or backticks ( ).

Example:
let firstName = "Paurnima";
let lastName = "Bawane";
let fullName = firstName + " " + lastName;
console.log(fullName);

Output:
Paurnima Bawane

* Number

• A Number in JavaScript is a data type used to represent numeric values, whether integer or decimal (floating-point).

• It is used for performing mathematical calculations like addition, subtraction, multiplication, etc.

Example:
let price = 99.99;
console.log(price);

Output:
99.99

* Bigint

• A BigInt in JavaScript is a special numeric data type used to represent very large integers that cannot be handled by the regular Number type.

Example:
let num = 20;
let bigNum = 50n;
let total = BigInt(num) + bigNum;
console.log(total);

Output:
70n

* Boolean

• A Boolean in JavaScript is a data type that can have only two values — true or false.

• It is mainly used for decision-making and conditional statements (like if, else).

Example:
let isLoggedIn = true;

if (isLoggedIn) {
  console.log("Welcome, user!");
} else {
  console.log("Please log in.");
}

Output:
Welcome, user!

* Undefined

• Undefined is a primitive data type in JavaScript.

• It means a variable has been declared but no value has been assigned to it yet.

Example:
let city = undefined;
console.log(city);

Output:
undefined

* Object

An Object in JavaScript is a non-primitive data type used to store multiple values in the form of key–value pairs.

Example:
let student = {
name: "Riya",
rollNo: 45,
subjects: ["Math", "English", "Science"],
isPassed: true
};
console.log(student);

Output:
{
  name: "Riya",
  rollNo: 45,
  subjects: ["Math", "English", "Science"],
  isPassed: true
}

* Null

• Null is a primitive data type in JavaScript that represents an intentional absence of any value.

• It means the variable is empty or has no value assigned on purpose. 

Example:
let data = null;
console.log(data);

Output:
null

* Symbol

• A Symbol is a primitive data type in JavaScript that is used to create unique and immutable identifiers (values that are guaranteed to be different from any other).

• Symbols are often used as object property keys to avoid naming conflicts — because every Symbol is completely unique, even if they have the same description.

Example:
let a = Symbol("user");
let b = Symbol("user");
console.log(a === b);

Output:
False
